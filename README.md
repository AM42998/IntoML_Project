# Horse Colic Survival Prediction (ML Project)

This project uses the Horse Colic dataset from the UCI Machine Learning Repository to build a supervised classification model that predicts whether a horse with colic will survive.

## Project Structure
- data/ — raw dataset files
- notebook/ — Jupyter notebook with analysis
- models/ — saved ML models
- app/ — (optional) web app demo
- README.md — project documentation

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Open the notebook:
   jupyter notebook notebook/horse_colic_analysis.ipynb

## Dataset Source
UCI Machine Learning Repository: https://archive.ics.uci.edu/dataset/54/horse+colic
